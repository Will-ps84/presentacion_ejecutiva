Sección 01 // Estrategia

Sincronización Digital SCM para Perufarma S.A.

Perufarma S.A. gestiona un portafolio complejo con altos estándares de servicio exigidos por supermercados y farmacias del país. Esta propuesta de negocio presenta la **implementación de un Agente AI** para transformar nuestro proceso S&OP, optimizando el capital de trabajo de marcas representadas (como el cuidado capilar de *Every Strand*) y asegurando el Fill Rate a nivel nacional.

Protección de Ingresos

Evitamos rupturas de stock en puntos clave de venta para garantizar el flujo continuo de facturación en canales modernos.

Liberación de Caja (Cash Release)

Ajustamos dinámicamente los buffers de inventario para liberar capital de trabajo atrapado en mermas o sobrestocks provinciales.

Sección 03 // Capacidades SCM

¿Por qué un Agente AI en lugar de BI Estático?

A diferencia de un dashboard de Power BI tradicional (que solo muestra datos históricos pasados), el Agente AI razona, prescribe planes de acción y automatiza tareas.

Análisis Cognitivo y Tolerancia

Entiende variaciones semánticas en la data. No requiere que tus CSVs de Perufarma estén perfectamente limpios; el agente los unifica de forma inteligente.

Capacidad Prescriptiva de Red

No solo avisa del quiebre; calcula y prescribe la transferencia exacta de inventario (rebalanceo) más eficiente entre Lurín, Arequipa y Trujillo.

Delegación Descentralizada

Agrupa los códigos desprotegidos por el planner responsable de la marca y dispara alertas directas con el cuadro resumen por correo.

Seguridad Local en VS Code

El código corre de manera segura en la red interna de Perufarma. La información de compras y ventas nunca sale de tu servidor local.

Sección 06 // Distribución Regional

Descentralización Multi-CD de Perufarma

Para lograr una distribución ágil, el stock debe analizarse por punto geográfico. Rompemos la consolidación macro para evaluar la disponibilidad de manera aislada en cada nodo.

CD Lurín (Lima - Hub Central)

Puerto de ingreso nacional. Centraliza el stock de seguridad de importaciones, el cross-docking y coordina la distribución troncal interprovincial.

CD Arequipa (Hub Sur)

Garantiza el nivel de servicio para el mercado del sur del país. Su abastecimiento y coberturas se optimizan de forma independiente para evitar quiebres.

CD Trujillo (Hub Norte)

Punto de abastecimiento clave para la costa norte. Monitoreado preventivamente para evitar quiebres ante interrupciones de tránsito en la Panamericana.

Sección 08 // Eficiencia de Distribución

Modelo Prescriptivo: Lean Rebalancing ⚡

El motor analítico de Perufarma optimiza el inventario de forma prescriptiva. Al detectar quiebres locales y sobrestocks en otros CDs, el sistema gatilla propuestas automáticas de transferencia interprovincial.

Eliminación de Costo Comercial

Se aprovecha el inventario excedente de la propia red de Perufarma S.A., evitando la colocación de órdenes de compra spot de emergencia al extranjero.

Reducción de Sobrecostos de Flete

La planeación y visibilidad de tránsitos permite consolidar despachos interprovinciales ordinarios, erradicando los fletes express urgentes.
